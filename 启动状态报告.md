## 🔧 需要您完成的步骤

### 步骤 1：安装并启动 MySQL

如果还没有安装 MySQL，请先安装：
- Windows: 下载 MySQL Installer 或使用 XAMPP/WAMP
- 确保 MySQL 服务正在运行

**检查 MySQL 服务：**
```powershell
# 检查 MySQL 服务状态
Get-Service -Name "*mysql*"

# 如果服务未运行，启动它（需要管理员权限）
net start MySQL80
# 或
net start MySQL
```

### 步骤 2：创建数据库

打开 MySQL 命令行或使用 MySQL Workbench：

```sql
CREATE DATABASE dorm_power_guard CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

**或者使用命令行：**
```powershell
mysql -u root -p
# 然后输入密码，执行上面的 CREATE DATABASE 语句
```

### 步骤 3：配置数据库密码

编辑文件：`backend\.env`

找到这一行：
```env
DB_PASSWORD=your_password
```

**修改为您的实际 MySQL root 密码：**
```env
DB_PASSWORD=你的实际密码
```

**完整的数据库配置示例：**
```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的MySQL密码  # ⚠️ 这里必须修改！
DB_NAME=dorm_power_guard
```

### 步骤 4：重新启动后端

配置完成后，重新启动后端服务：

```powershell
cd c:\GitCloneRepository\dorm-power-guard-lite\backend
python run.py
```

**或者使用启动脚本：**
```powershell
cd c:\GitCloneRepository\dorm-power-guard-lite\backend
.\start.ps1
```
